## Project Overview
This project simulates a production-style workforce dataset generation pipeline used for analytics, internal reporting, and dashboard systems.

# Workforce Data Automation Pipeline

Python-based data engineering project that generates large-scale synthetic HR datasets for analytics, reporting, and dashboard systems.

## Features
- Generates 8,950+ employee records
- Uses Faker for realistic data simulation
- Applies probability distributions for departments, salaries, and hiring
- Performs data transformation using Pandas
- Outputs structured CSV for downstream analytics

## Tech Stack
Python, Pandas, NumPy, Faker

## How to Run

pip install -r requirements.txt  
python data_pipeline.py

## Output
HumanResources.csv containing structured workforce data.
